## 📦 Getting Started
To run this project locally:

```bash
# Clone the repository
git clone https://github.com/Arun0039/arunk07.git
cd arunk07

# Install dependencies
npm install

# Run the development server
npm run dev
```

Visit `http://localhost:3000` to view the site.
